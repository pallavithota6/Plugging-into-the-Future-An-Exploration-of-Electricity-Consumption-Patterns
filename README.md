# Plugging into the Future: An Exploration of Electricity Consumption Patterns

## Project Overview

This project analyzes electricity consumption patterns across different states, regions, metro cities, months, quarters, and years using MySQL, Tableau, and Flask. The objective is to provide meaningful insights through interactive dashboards and visualizations.

---

## Objectives

- Analyze electricity consumption trends.
- Compare electricity usage across different regions and states.
- Identify top and bottom electricity-consuming states.
- Study monthly, quarterly, and yearly consumption patterns.
- Create interactive dashboards and stories using Tableau.
- Publish dashboards to Tableau Public.
- Integrate Tableau Dashboard into a Flask web application.
